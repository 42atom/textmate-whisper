# textmate-whisper v0.2.1

## Fixes
- Fork build now uses unique bundle identifier:
  - `com.textmatewhisper.TextMate`
- Avoids permission collision with official `/Applications/TextMate.app` (`com.macromates.TextMate`).
- Release script now prefers:
  - `~/Desktop/textmate-whisper-build/TextMate-Whisper.app`
  - falls back to `TextMate.app`

## Downloads
- `TextMate-whisper-macos-universal-v0.2.1.zip`
- `TextMate-whisper-macos-universal-v0.2.1.zip.sha256`

## Notes
- If both official TextMate and fork build are installed, use `TextMate-Whisper.app` for voice recording tests.
