# textmate-whisper v0.2.0

## Highlights
- Voice dictation command set simplified to toggle-based workflow:
  - `Option+Command+F1`: toggle recording
  - `Option+Command+F2`: force stop fallback
- Added microphone permission helper command in bundle menu.
- Added TextMate fork assets for microphone permission fix:
  - patch file
  - build script skeleton
  - research + plan docs
- Added release script to publish compiled app artifacts.

- Build validated on macOS with Xcode 26.2 and Homebrew dependencies.
